Week 5: ~~~

This week I learned about how a web browser unfortunately does not hold every font in the world, and how we must therefore use fallback fonts in case the user does not have any nice ones. Nonetheless, I will stick with nice fonts as my main, I think. Found out how to use google fonts, too, after a little bit of a misspelling issue. I then learned how to better style my text, with things like indenting - which I did add - and learned how margins merge. What was really nice is that I found out how to add those giant quote marks to a blockquote - wooo! Finally, I kinda learned how float works - still cannot get it to properly float without having the first line of text above the image. Oh well.

I learned to:

  1. More correctly define fonts.
  2. Use web fonts.
  3. Style my text all nice.
  4. Float images, kind of.

Week 4: ---

This week I learned how to attach CSS to a web page, and to do some basic CSS, as covered in the tutorial, such as color, font-family, margin, font-size. I also learned about how we can measure a web page in pixels, and therefore change visuals based on that. Especially important is that I learned how we can use id and class to give specific styles to individual or grouped elements - such as using a wrapper. I also learned how to make my links pretty, and how I could validate the CSS. Turned out just fine.

I learned to:

  1. Attach CSS to a page.
  2. Alter said CSS, and create my own on a basic level.
  3. Style with Id and Class.
  4. Validate CSS for a website.
